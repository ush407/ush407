CSS 정리

CSS : 작성 위치

  1. 인라인 (body 영역) 위치
  2. 내부파일 (head 영역) 위치
  3. 외부파일 ( *.css) 위치

CSS : 셀렉터

  1. 태그 셀렉터 : html태그명
  2. 아이디 셀렉터 : #
  3. 클래스 셀렉터 : .
  4. 가상클래스 셀렉터 : 예-:hover
  5. 자식 셀렉터 : 셀렉터 > 셀렉터
  6. 자손(후손) 셀렉터 : 셀렉터 셀렉터

CSS : 텍스트 관련

  1. 색상 적용 : color
  2. 정렬 및 줄 간격 지정 : text-align, text-decoration, line-height
  3. 글자에 그림자 넣기 : text-shadow
  4. 글자의 글꼴과 크기, 스타일, 두께 지정 : font-family, font-size, font-style, font-weight
    ▶ font단축

CSS : 박스 모델과 디스플레이

  1. 경계선 그리기 : border
    • 단락에 경계선
    • 다양한 형태의 경계선
  2. 박스의 모서리, 박스 그림자 설정 : border-radius, box-shadow
  3. 박스 바깥쪽 마진 여백 : margin
  4. 기본 설정된 여백 초기화
